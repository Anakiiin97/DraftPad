# DraftPad

Un outil de rédaction léger et rapide, fonctionnant entièrement dans le navigateur.

### Fonctions clés :
* 🎯 **Mode Focus** pour une concentration totale.
* 🌙 **Mode Nuit** pour le confort visuel.
* 📋 **Panneau de notes** pour du brouillon.
* 🖱️ **Drag & Drop** pour organiser vos textes.
* 💾 **Zéro Cloud** : Tout est stocké localement dans votre navigateur (vie privée respectée).
